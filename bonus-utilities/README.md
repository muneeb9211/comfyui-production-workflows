# Bonus Utility Workflows

Small, single-purpose workflows for common tasks.

| Workflow | Nodes | What It Does |
|---|---|---|
| `audio-separator.json` | 4 | Separate vocals from instrumentals (AudioSeparateVocals) |
| `caption-generator.json` | 5 | Auto-caption image datasets using WD14 Tagger |
| `voice-design-tts.json` | 4 | Design custom voices with Qwen3-TTS |
| `video-combine.json` | 8 | Merge multiple video clips with audio concatenation |
| `singing-audio-merge.json` | 8 | Amplify and combine audio tracks for final mix |

These are intentionally simple — drag, drop, and run.
